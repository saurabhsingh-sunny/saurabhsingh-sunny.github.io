# Technology Icons Download Guide

## Download Official SVG Icons

### 1. Python (Official Blue/Yellow Logo)
**Source:** https://www.python.org/static/community_logos/python-logo-master-v3-TM.png
**Better SVG:** https://github.com/devicons/devicon/blob/master/icons/python/python-original.svg
- Right-click "Raw" → Save as `python.svg`

### 2. R
**Source:** https://github.com/devicons/devicon/blob/master/icons/r/r-original.svg
- Right-click "Raw" → Save as `r.svg`

### 3. Tableau
**Source:** https://github.com/devicons/devicon/blob/master/icons/tableau/tableau-original.svg
**Or:** https://logos-world.net/tableau-logo/ (download SVG version)
- Save as `tableau.svg`

### 4. Power BI
**Source:** https://github.com/microsoft/PowerBI-Icons (official Microsoft repo)
**Or:** https://upload.wikimedia.org/wikipedia/commons/c/cf/New_Power_BI_Logo.svg
- Save as `powerbi.svg`

### 5. MySQL/SQL
**Source:** https://github.com/devicons/devicon/blob/master/icons/mysql/mysql-original.svg
- Save as `mysql.svg`

### 6. VS Code
**Source:** https://github.com/devicons/devicon/blob/master/icons/vscode/vscode-original.svg
- Save as `vscode.svg`

### 7. GitHub
**Source:** https://github.com/logos (official GitHub logos)
**Or:** https://github.com/devicons/devicon/blob/master/icons/github/github-original.svg
- Save as `github.svg`

### 8. Quarto
**Source:** https://quarto.org/quarto.png (PNG)
**SVG:** Custom or convert from PNG
- Save as `quarto.svg` or `quarto.png`

## Quick Download All from Devicon

Visit: https://github.com/devicons/devicon/tree/master/icons
- Navigate to each technology folder
- Download the `-original.svg` version for colored official logos
- Save to this `/images/icons/` folder

All icons should be saved as SVG format with their original colors intact.
